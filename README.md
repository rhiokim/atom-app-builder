# Atom App Builder based on Node.js & Grunt.js

## requirements

* node.js
* grunt-cli

## How to install

1. clone

```bash
$ git clone git@github.com:rhiokim/atom-app-builder
```

2. install

```bash
$ cd /path/to/atom-app-builder
$ npm install
```

3. setup atom-shell

```bash
$ grunt down --atom=x.x.x
```

## Create your app

1. create new app

```bash
$ grunt new --app=myApp
```

## Run your app

1. run myApp

```bash
$ grunt run --app=myApp
```

## Update atom-shell

1. download bin

```bash
$ grunt down --atom=0.x.x
```
